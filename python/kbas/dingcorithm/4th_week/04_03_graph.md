# 그래프란?

> 연결되어 있는 점 간의 관계를 표현하는 자료구조

이를 표현하는 방법에는 두 가지가 있다.

1. 인접행렬
2. 인접리스트

## 1. 인접행렬

아래와 같이 0번 노드가 1번 노드와 연결되어있다는 등의 관계를 2차원 배열로 표현할 수 있다.

```python
graph = [
    [False, True, False, False],
    [True, False, True, False],
    [False, True, False, True],
    [False, False, True, False],
]
```

이는 무언가를 조회하는 데에는 시간복잡도가 O(1)이다.
ex) graph[1][2]

## 2. 인접리스트

아뢔 같이 모든 노드에 연결되 노드에 대한 정보를 차례대로 저장한다.

```python
graph = {
    0: [1],
    1: [0, 2],
    2: [1, 3],
    3: [2]
}
```

근데 얘는 무언가를 조회할 때 시간복잡도가 O(N)이 걸린다.

## 정리

> 시간 vs 공간

### 시간

- 인접 행렬은 O(1)
- 인접 리스트는 O(N)

### 공간

- 인접행렬은 시간은 많이 절약하지만, 공간을 N^2 만큼 차지 한다.
- 인접 리스트는 현재 노드들의 개수 \* 간선의 개수 만큼만 차지한다. 아무리 커져도 제곱만큼 커지지는 않는다. 아무리 해봤다 O(N)정도다 라고 생각하면 된다.

> 확보하고자 하는 것에 따라 알고리즘을 선택하면 된다.
